# Superstore Sales Analysis Dashboard

## Overview
A Power BI dashboard analyzing US Superstore sales data, examining performance 
across regions, products, customer segments and shipping modes to derive 
actionable business insights.

## Problem Statement
Without proper visualization, patterns in sales, profit and customer behavior 
across regions and product categories are difficult to identify. This dashboard 
provides a comprehensive view of business performance to support data-driven 
decision making.

## Tools Used
- Power BI Desktop
- Power Query (Data Cleaning)
- DAX (Measures)

## Dataset
- 9,994 orders across the United States
- Features: Sales, Profit, Discount, Region, Category, Segment, Ship Mode

## Dashboard Pages
1. **Sales Overview** - KPIs, sales trends, category distribution
2. **Regional Performance** - Sales and profit by region, top 10 states, map
3. **Product Analysis** - Category and sub-category performance, discount impact
4. **Customer Segments** - Sales and profit by segment, order distribution
5. **Shipping Analysis** - Ship mode performance, trends, profit distribution

## Key Insights
- West region leads in both Sales ($0.73M) and Profit ($108K)
- Technology is the highest revenue category
- Standard Class is the most used shipping mode (57% of profit)
- Consumer segment generates the highest sales across all months
